# 🌐 Frontend Studies

Repositório dedicado ao aprendizado de desenvolvimento frontend, começando com HTML e evoluindo para CSS, layouts responsivos e integração com backend.

---

## 📚 Conteúdo

Este repositório inclui:

- Estrutura básica de páginas HTML
- Uso de headings (`h1` até `h6`)
- Parágrafos (`p`)
- Imagens (`img`)
- Organização inicial de conteúdo
- Estilização com CSS

---

## 🚀 Objetivo

Desenvolver uma base sólida em frontend para:

- Criar interfaces bem estruturadas
- Entender semântica HTML
- Trabalhar com CSS e design responsivo
- Integrar com backend (PHP)

---

## 🛠️ Tecnologias

- HTML5
- CSS3
